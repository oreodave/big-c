A dynamic multiplayer playing card game, written in Rust.

Goals:
- Server/client setup for multiplayer games.
- Allow up to 12 players to interact in any one game.
- Player ranking system.

Requirements:
- Rust

Run:
    $ cargo build
to compile the program.
